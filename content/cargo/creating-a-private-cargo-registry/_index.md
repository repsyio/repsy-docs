title = "Creating a Private Cargo Registry"
weight = 61
+++

Before creating a registry, you need to [register to Repsy](https://repsy.io/) with a username, password, and e-mail. When you register, Repsy will create a `cargo` registry for you and will publish a sample crate under it! This is how fast you can get started.

In the Cargo section, you can create new registries and change the settings for existing ones whenever you need.
